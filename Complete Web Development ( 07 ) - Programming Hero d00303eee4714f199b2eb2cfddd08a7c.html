  • CSS Animation Cheatsheet:
/* Animation shorthand */
animation: name duration timing-function delay iteration-count direction fill-mode play-state;

/* Animation properties */
animation-name: none;
animation-duration: 0s;
animation-timing-function: ease;
animation-delay: 0s;
animation-iteration-count: 1;
animation-direction: normal;
animation-fill-mode: none;
animation-play-state: running;

/* Example */
animation: bounce 1s linear 0s 3 alternate;

Here is what each of the properties does:

  • animation-name: The name of the @keyframes animation.
  • animation-duration: The length of time it takes for the animation to complete one cycle.
  • animation-timing-function: Controls the speed curve of the animation.
  • animation-delay: The amount of time to wait before starting the animation.
  • animation-iteration-count: The number of times the animation should be played.
  • animation-direction: The direction in which the animation should be played.
  • animation-fill-mode: Determines how the animation styles are applied to the element before and after the animation.
  • animation-play-state: Determines whether the animation is running or paused.

  • Repetition: যখন একসাথে অনেকগুলো জিনিস থাকবে তখন আমাদের ensure করতে হবে যে, যেই জিনিস গুলো repeat হচ্ছে তাদের height, width, content যেন same হয়। আর না হলে সেটা design principle break করবে। ফলে, আমারদের design দেখতে বাজে দেখাবে।

javascriptCopy code

In this example, we have an array called numbers. We can use the apply() method to call the Math.max() function with the numbers array as the arguments. The apply() method takes null as its first argument, since we don't need to specify a this value for the Math.max() function.
