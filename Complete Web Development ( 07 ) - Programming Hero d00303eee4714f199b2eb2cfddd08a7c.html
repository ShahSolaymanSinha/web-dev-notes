Here is what each of the properties does:

  • animation-name: The name of the @keyframes animation.
  • animation-duration: The length of time it takes for the animation to complete one cycle.
  • animation-timing-function: Controls the speed curve of the animation.
  • animation-delay: The amount of time to wait before starting the animation.
  • animation-iteration-count: The number of times the animation should be played.
  • animation-direction: The direction in which the animation should be played.
  • animation-fill-mode: Determines how the animation styles are applied to the element before and after the animation.
  • animation-play-state: Determines whether the animation is running or paused.

‚≠êÔ∏è Here's a JavaScript DOM (Document Object Model) cheat sheet that covers some of the basics:

  1. Selecting elements:
  • document.getElementById(id): selects an element by its unique ID
  • document.getElementsByClassName(className): selects elements by their class name
  • document.getElementsByTagName(tagName): selects elements by their tag name
  • document.querySelector(selector): selects the first element that matches a CSS selector
  • document.querySelectorAll(selector): selects all elements that match a CSS selector

  1. Modifying elements:
  • element.innerHTML: sets or gets the HTML content inside an element
  • element.textContent: sets or gets the text content inside an element
  • element.style: sets or gets the inline style of an element
  • element.setAttribute(attribute, value): sets the value of an attribute on an element
  • element.getAttribute(attribute): gets the value of an attribute on an element
  • element.classList.add(className): adds a class to an element
  • element.classList.remove(className): removes a class from an element
  • element.classList.toggle(className): toggles a class on an element

  1. Creating and inserting elements:
  • document.createElement(tagName): creates a new element with the specified tag name
  • element.appendChild(childElement): adds an element as a child of another element
  • element.insertBefore(newElement, referenceElement): inserts an element before another element

  1. Removing elements:
  • element.remove(): removes an element from the DOM
  • element.parentNode.removeChild(element): removes an element from its parent node

  1. Traversing the DOM:
  • element.parentNode: gets the parent node of an element
  • element.childNodes: gets the child nodes of an element
  • element.nextSibling: gets the next sibling node of an element
  • element.previousSibling: gets the previous sibling node of an element
  • element.firstChild: gets the first child node of an element
  • element.lastChild: gets the last child node of an element.

This cheat sheet should give you a good starting point for working with the DOM in JavaScript. Remember, the DOM is a dynamic structure, and you can modify it in real-time with JavaScript to create dynamic and interactive web pages.
